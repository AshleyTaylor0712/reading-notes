
# Class 04 Reading Notes

## Wireframe

- Used to plan the layout of website as well as the path users will follow to access information within the product.
- Wireframes can be drawn by hand or built using software.

### Wireframe Tools

- Pen & Paper
- UxPin: Clickable Prototypes
- InVision: Clickable Prototypes & Feedback Enabled
- Wireframe.cc: Geared towards quick creation

### Build a Wireframe

1. Reasearch: target audience/users, competitors, trends, best practices & project guidelines
2. Documents & Organize research
3. Plan out a streamlined user flow 
4. Make an undetailed draft that is less focused on design and more function & flow orientated
5. Add functionality details ex. navigation & calls to action
6. Convert Wireframe into a Prototype
