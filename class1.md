## Markdown

### Headings:

- (#) Controls heading size
- Include a space between (#) & text
- Place a blank line before and after heading

### Paragraphs

- Use a blank line to separate
- Don't indent paragraphs

### Line Break

- place two spaces at end of line to make a line break

### Emphasis

- Bold: place (**) on both sides of text 
- Italic: Place (*) on both sides of text
- Bold & Italic (***) on both sides of text

### Blockquote

- Create new paragraph & place a (>) + space before text.   
- Multiple Paragraphs: add a (>) on the blank lines between the paragraphs.
- Create nested blockquotes by placing a (>>) in front of paragraph.
- Blockquotes can contain other formatting elements such as headings, lists and text adjustments.
- **Place blank lines before and after blockquotes.**

Example:

> Block quote dislapy.
>
> Multi Paragraph Format.
> 
> > Nested Paragraph Example.
>
> #### Blockquote Heading Example
> 
> - Blockquote **List Example**
> - Blockquote *List Example*

### Lists

- **Ordered Lists** Add numbers with periods to line items
- **Unorderd Lists** Add a (-), (*) or (+) before line items
  - Indent to create a nested list
  - Ordered lists can be nested under unordered lists (vice versa)
  
> Note: Use a backslash to start a unordered list with a number followed by a period.
>
>> Example: 1968(backslah).

### Images

- Add images using this format:
  - ! [ **Text Displayed** ]( *link to image* )

### Links

- Add links using this format:
  - [ **Text Displayed** ] ( *link to image* )
- Link titles can be added by encloseding in ("") after url
  - Example: [ **Text Displayed** ] ( *link to image* "link title" )

### Emojis

- Typing ":" will bring up a list of emojis that can be used

### Footnotes

- Add a footnote by adding a ([^1])
