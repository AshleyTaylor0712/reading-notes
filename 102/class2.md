## Text Editor

### What is it?

Software that eases and enables the task of writing the code that will be used to build a website.

#### Important Features:

- Extension availibility
  - Features expanstion through plugins
- Code Completion
  - Displays suggestions once typing begins
  - Prroduces closing tags
- Syntax Highlighting
  - Color codes attributes, elements 
- Theme Adjustment
  - Allows you to adjust background color to reduce eye strain

## Terminal 

### The Command Line:

A text based interface that can be fed commands. Terminal can be opened via applications.

- Line 1: Prompt & Command
- Line 2-5: Output as a response to command.
- Line 6: Prompt & Command

### A Shell:

- Dictates terminal behavior & apperance
- Bash is the most commonly used

### Navigation:

## Commands:

- **pwd** (Print Working Directory): Displays current working directory
- **ls** (list): Displays current Location

## Paths:

A path is used to access a file or directory.

- Absolute: Pulls files based on root directory
> Root: Top of directory. Annotated by a (/).
- Relative: Pulls files based on users current location in system.
